# macOS 风格计算器 - 微信小程序

一个基于 macOS 原生设计风格的计算器微信小程序，采用玻璃态（毛玻璃）效果，提供完整的计算功能和优雅的用户体验。

## 特性

- **macOS 原生设计** - 圆角、配色、字体遵循 macOS 设计语言
- **玻璃态效果** - 毛玻璃背景、透明层次
- **全屏布局** - 隐藏导航栏，计算器充满屏幕
- **完整功能** - 基础运算、正负切换、百分比、链式计算
- **浮点精度修正** - `0.1 + 0.2 = 0.3`
- **除零保护** - 除以零显示"错误"
- **自适应显示** - 长数字自动缩小字体

## 项目结构

```
├── app.js                # 小程序入口文件
├── app.json              # 小程序配置文件
├── app.wxss              # 全局样式文件
├── project.config.json   # 项目配置文件
├── sitemap.json          # 站点地图配置
└── pages/
    └── index/
        ├── index.wxml    # 页面结构
        ├── index.wxss    # 页面样式
        ├── index.js      # 页面逻辑
        └── index.json    # 页面配置
```

## 快速开始

1. 克隆项目
2. 打开微信开发者工具，导入项目文件夹
3. 填写 AppID（或使用测试号）
4. 在模拟器中预览

## 声明

本项目初始版本由 [OpenClaw](https://openclaw.ai) AI 助手生成，后经 [Claude Code](https://claude.ai/claude-code) 二次开发优化，主要改进包括：

- 移除硬编码演示数据，初始状态归零
- 移除假电池动画和演示性 UI 元素

- 修复除零返回 0 的问题，改为显示"错误"
- 修复浮点精度问题（`parseFloat(result.toFixed(10))`）
- 修复链式计算逻辑
- 添加 `app.js` 入口文件
- 全屏自定义导航栏布局
- 计算器状态改为实例属性，减少不必要的响应式开销
- 合并多次 `setData` 调用，优化渲染性能
- 添加输入长度限制和结果自适应字体
