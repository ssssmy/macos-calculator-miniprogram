# macOS 风格计算器 - 微信小程序


### 玻璃态效果
```css
.glass-effect {
  background: rgba(255, 255, 255, 0.15);
  backdrop-filter: blur(40rpx);
  border: 2rpx solid rgba(255, 255, 255, 0.2);
  box-shadow: 0 40rpx 120rpx rgba(0, 0, 0, 0.3);
}
```

### 响应式设计
- 使用 `rpx` 单位适配不同屏幕
- 媒体查询优化移动端体验
- 按钮尺寸自适应

## 📱 适配说明

### 屏幕尺寸
- iPhone 6/7/8: 375×667
- iPhone X/11/12: 375×812
- iPad: 768×1024
- 安卓设备: 多种尺寸

### 单位系统
- 使用 `rpx` 作为主要单位
- 1rpx = 屏幕宽度/750
- 完美适配不同DPI设备
